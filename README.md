# Advanced Calculator

Wie gesagt habe ich einen besseren Taschenrechner gemacht – jetzt ist es viel besser designed mit mehreren Funktionen. Mit diesem Taschenrechner kann man eigentlich das Gleiche machen wie bei einem normalen handhablichen Taschenrechner. Mein Comeback nach der ersten Version! 🧮🔥

Ein advanced Web-Calculator mit wissenschaftlichen Basics, sauberem Layout und mehr Komfort – fast wie ein echter Taschenrechner aus der Tasche. Noch nicht perfekt, aber ein riesiger Sprung: besseres Design, neue Features und alles, was ich mir vorgenommen hatte.

### Technologies
- HTML
- CSS
- JavaScript

Alles vanilla – pure HTML/CSS/JS, kein Framework, kein Build-Tool, direkt in VS Code gebaut und auf Netlify live.

### Features
- Grundrechenarten: + − × ÷
- Dezimalpunkt (.)
- AC-Button (Alles löschen)
- = zum Berechnen
- Wurzel (√)
- Pi (π)
- Klammern ( )
- Quadrat/Hochzahl (x²)
- Hover-Effekte auf Buttons (werden heller beim Drüberfahren)
- Zeigt "Infinity" bei Division durch 0
- Responsive – funktioniert gut auf Handy
- **WICHTIG: # Button** – Mit dem Knopf "#" kannst du alle neuen/advanced Features öffnen (Wurzel, Pi, Dezimal, Hochzahl usw.) – Toggle für den erweiterten Modus!
- Verlauf: Es vergisst nicht, was du vorher geschrieben hast (läuft einfach weiter im Display), außer du löschst es – aber kein Zugriff auf alte Rechnungen
- Keine Tastatur-Eingabe, keine Dark Mode, keine Extra-Animationen außer Hover

### The Process
Ich wollte einen Comeback machen, weil der erste schlecht war – das Design war nicht so toll und es fehlten viele Funktionen. Jetzt hat es eigentlich alles, was ich machen wollte.  

Am schwierigsten war JavaScript: Es war schwierig zu verstehen, wie man das alles anpasst, und vor allem weil ich nicht so begabt bin mit JavaScript.  

Dabei habe ich neue basic Funktionen in JS gelernt – z. B. wie man mit einem Knopfdruck einen Übergang/Modus-Wechsel macht (wie beim # Button).  

Ich bin jetzt richtig happy, dass ich geschafft habe, was ich erreichen wollte, und es gefällt mir, dass ich schon so vieles schaffe!
